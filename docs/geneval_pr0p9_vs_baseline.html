关键洞察

1. 整体退化仅 ~1.08 pp：在保留 ~10% 候选 token（剪枝率 0.9）极激进的设置下，GenEval 整体准确率仅下降约 1 个百分点，>84% 的 prompt 评估完全不变。
2. 多数 regression 是 detector 边缘噪声：7 个完全翻转 case 中，至少 5 个（00163, 00178, 00477, 00492, 00495）的 baseline 通过依赖 detector 在 0.3~0.5 临界 confidence 的检测。Pruning 引入的微小图像扰动恰好与 detector 决策边界相交，导致 ±1~3 张图的随机翻转。
3. 真正的语义损伤集中在两类：
  • Counting · 多实例边界平滑：3+ 个相邻同类实例的边界被剪枝平滑，detector 计数不准（典型：00217 三本书、00254 四把椅子）。
  • Color attribute · 浅色染色：white / yellow 等浅色物体易被周围色调"染色"导致颜色判定错误（典型：00321 white sheep 被草色渗染、00477 brown bed 颜色变弱）。
4. 类别鲁棒性排序： colors ≈ single_object > counting ≥ two_object > position > color_attr
5. 工程建议：
  • 对 counting / color_attr 任务降低 pruning ratio 至 0.7~0.8 以保留细节 token；
  • 对 single_object / colors / position 等鲁棒类别保持 0.9 极激进剪枝；
  • 论文中将上述两类失败模式作为已知 trade-off 明确报告，并展示视觉对比的微小性。
